# Hospital Advertisement Portfolio

This project is a frontend portfolio for a hospital advertisement. It showcases the hospital's services, promotional content, and contact information through a user-friendly interface.

## Project Structure

```
hospital-portfolio
├── src
│   ├── assets          # Static assets such as images, fonts, and stylesheets
│   ├── components      # Reusable components
│   │   └── Header.tsx  # Header component with logo and navigation
│   ├── pages           # Application pages
│   │   └── Home.tsx    # Main landing page for the hospital
│   ├── App.tsx         # Main application component
│   └── main.tsx        # Entry point of the application
├── public
│   └── favicon.svg     # Favicon for the application
├── index.html          # Main HTML file
├── package.json        # npm configuration file
├── tsconfig.json       # TypeScript configuration file
└── README.md           # Project documentation
```

## Getting Started

To get started with the project, follow these steps:

1. **Clone the repository:**
   ```
   git clone <repository-url>
   cd hospital-portfolio
   ```

2. **Install dependencies:**
   ```
   npm install
   ```

3. **Run the application:**
   ```
   npm run dev
   ```

4. **Open your browser:**
   Navigate to `http://localhost:3000` to view the application.

## Features

- Responsive design for optimal viewing on various devices.
- Easy navigation through the header component.
- Promotional content highlighting the hospital's services.
